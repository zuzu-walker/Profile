A Dummy's Guide to Coding

So you don't feel out of the loop


Coding can seem daunting. But with these small tips, you can begin your journey into understanding it. We will go over how a html page is formatted, and how to access the code on a website on Google Chrome. For some inspiration, head to the next page for some iconic people who have contributed to computer science!

1. How to find code on a website

To find the code to a website, you need to first open a webpage you want to inspect. Once you've done this, right click, and select 'Inspect' on the options. There you go! You've now found out how to open the code on a webpage.

2. Structure of HTML

The structure of html is a lot easier thank you think! Everything needs to lie inbetween 'tags'. Each tag will perform a different function e.g. h1 is a header, p is a paragraph etc. Some tags need tags within them, like images, and if you need help figuring out how to use a specific tag fill out our form at the bottom, and we will get back to you!
